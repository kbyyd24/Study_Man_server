功能接口：
register:（注册）
--------------------------------------------------------------------------------
| name			|用户名，不可与其他用户名重复
--------------------------------------------------------------------------------
| password		|密码，最好加密后上传
--------------------------------------------------------------------------------
返回数据：
<xml version='1.0' encoding='UTF-8'>
<root>
	<id>					//用户id，注册失败id为0
	<result>				//注册结果或失败原因
</root>

loginer:（登陆）
--------------------------------------------------------------------------------
| name			|用户名
--------------------------------------------------------------------------------
| password		|用户密码
--------------------------------------------------------------------------------
返回数据：
<xml version='1.0' encoding='UTF-8'>
<root>
	<id>					//用户id，登陆失败id为0
	<result>				//登陆结果或失败原因
</root>

noteLoger:（上传笔记存入数据库）
--------------------------------------------------------------------------------
| userId		|用户id，登陆时从服务器获得
--------------------------------------------------------------------------------
| title			|笔记标题，不超过80个字
--------------------------------------------------------------------------------
| objectKey		|笔记在oss上存储的名称
--------------------------------------------------------------------------------
| sharing		|是否共享，true or false，空则为false
--------------------------------------------------------------------------------
| time			|笔记创建时间
--------------------------------------------------------------------------------
返回数据：
<xml version='1.0' encoding='UTF-8'>
<root>
	<result>				//笔记上传结果
</root>

noteLister:（笔记目录）
--------------------------------------------------------------------------------
| userId		|用户id
--------------------------------------------------------------------------------
| page			|页数
--------------------------------------------------------------------------------
| number		|每页问题数目，可以为空，默认为10
--------------------------------------------------------------------------------
返回数据：
<xml version='1.0' encoding='UTF-8'>
<root>
	<result>				//查询结果
	<note>
		<id>				//笔记id
		<title>				//笔记标题
		<time>				//笔记创建时间
	</note>
</root>

| id			|用户id

<xml version='1.0' encoding='UTF-8'>

	<objectKey>				//笔记在OSS上的名字

</root>

uploader:（上传）可以上传笔记，提问和回答
--------------------------------------------------------------------------------
| objectKey		|笔记、提问和回答在OSS上的位置
--------------------------------------------------------------------------------
| contentType	|设置content-Type
--------------------------------------------------------------------------------
| userId		|用户id，用于查询笔记或问题是否存在
--------------------------------------------------------------------------------
| title			|笔记或问题的标题，查询是否已经存在
--------------------------------------------------------------------------------
| type			|上传的类型，note,question or answer
--------------------------------------------------------------------------------
返回数据：
<xml version='1.0' encoding='UTF-8'>
<root>
	<result>				//上传的笔记或提问是否已存在
	<authorization>			//签名结果
	<bucket>				//bucket名字
	<objectKey>				//上传的笔记、提问或回答在OSS上的位置
	<date>					//签名时间
</root>
